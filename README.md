vim
===

My vim configuration

**Install:**

    cd
    git clone https://github.com/BenjaminRH/dotvim.git .vim
    cd .vim
    git submodule init
    git submodule update
    cd ..
    ln -s .vim/.vimrc

**Plugins to consider:**

 * vinegar (to replace NERDTree)
 * unimpaired
 * abolish
 * detectindent
